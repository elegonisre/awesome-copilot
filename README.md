## Copilot Instructions

Custom `.github/copilot-instructions.md` files that tailor GitHub Copilot behavior for specific stacks and workflows.

- **General Best Practices** — Universal coding standards and conventions
- **Python / Django** — Django project conventions, ORM patterns, and testing
- **TypeScript / React** — Component patterns, hooks, and type safety
- **Rust** — Memory safety idioms and cargo workspace conventions
- **Go** — Idiomatic Go patterns and error handling
- **Node.js / Express** — REST API conventions, middleware patterns, and async handling
- **Java / Spring Boot** — Spring conventions, dependency injection, and REST patterns *(personal addition)*
- **Terraform** — HCL conventions, module structure, and provider patterns *(personal addition)*
